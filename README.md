# Календарь праздников

Приложение на C# и WinForms для получения списка праздников по стране и году из открытого API [Nager.Date](https://date.nager.at/).

## Стек технологий
- C# .NET Framework
- Windows Forms
- HttpClient (асинхронные запросы)
- Newtonsoft.Json (десериализация JSON)
- REST API (Nager.Date)

## Функциональность
- Загрузка списка стран из API.
- Выбор страны и года.
- Получение списка праздников.
- Сохранение списка в текстовый файл (кнопка «Сохранить»).
- Очистка списка с автоматическим сбросом года на текущий (кнопка «Очистить»).

## Как запустить
1. Откройте решение в Visual Studio.
2. Установите пакет Newtonsoft.Json через NuGet.
3. Запустите приложение (F5).
